## Installation
Github:

    git clone https://github.com/I2rys/webgex
    
NPM Packages:
```
npm i axios
```

## Usage
```
node index.js <link> <output> <regex>
```

+ link - Target website link.
+ output - In where to save the results.
+ regex - The regex to use.
